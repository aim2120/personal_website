Personal Website Version 1

Screenshot of my workspace
  • Client: Myself
  • Date Completed: January 2016
  • Languages Used: HTML, CSS, JavaScript, Java
I created this website from scratch both to expand my online presence and to practice my web development skill. My university provides a server that I can SSH into to update a directory created for my university ID. Although I have used HTML, CSS, and JavaScript together before, this is the first time I've created a fully fleshed-out website. In Java I programmed an update runner that I used to modify and generate HTML and JavaScript text quickly. Currently the update runner can generate HTML image tags for my art page and JavaScript arrays for use in the image randomizer in the front page, update the copyright year on all pages, and find/replace a given String on all HTML pages present.

StarField Game

Click to play!
  • Client: iD Tech Camps
  • Date Completed: July 2015
  • Languages Used: PaperScript, HTML
I created this game as an example for an Intro to Scratch and JavaScript class I taught at iD Tech Camps' Columbia University program. The coursework was based in PaperScript, which allows for the easy creation of graphical programs. This game was created to illustrate the basic concepts of movement, collisions, timer, scoring, "start game"/"game over" screens, click-based actions, custom sprites, and changing backgrounds.
